# Operations Domain

This domain handles automated scripts, background cron jobs, browser automation (Puppeteer), database migrations, and infrastructure tasks.

**Rules for this domain:**
- Prioritize reliability and logging.
- Any new web scraping workflow must be documented here as a skill.
